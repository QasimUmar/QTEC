Introducing Git Lab - A Learning Project for Version Control

INTRODUCTION

In this project, you'll learn to use a distributed version control system called Git. You'll also learn to connect to a VM instance, install Git, configure your Git user information, and perform basic Git operations like adding files, editing files, and committing changes.

GitHub Project Repo

Lab Objectives

  • Create a Git repository
  • Add files to the repository
  • Edit the files
  • Commit the changes to the repository

Step-by-Step Instructions

Step 1: Install Git

Run the following command to install Git on your Linux VM:

sudo apt update
sudo apt install git

Verify the installation:

git --version

Step 2: Create a Git Repository

Create a directory for your Git repository:

mkdir my-git-repo
cd my-git-repo

Initialize a Git repository:

git init

Step 3: Configure Git User Information

Set up your username and email:

git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"

Step 4: Create and Commit Files

Create a file named README and add content:

nano README

Add and commit the changes:

git add README
git commit -m "Initial commit"

Step 5: View Commit Log

View the commit history of the repository:

git log
